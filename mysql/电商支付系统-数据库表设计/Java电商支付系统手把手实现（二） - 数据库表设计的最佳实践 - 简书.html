  • 乍一看，有必要搞这么多种的时间嘛？有以下诸多原因
    • 前端显示需要，那就必须存着呀！
    • 方便定位排查问题，比如某用户投诉某订单一直不发货，肯定就需要时间去定位
    • 方便数据分析，比如需要计算从用户支付到最终发出商品的平均时间
    • 根据订单状态确认相应订单时间

3 时间戳

用于定位排查业务问题

  • create_ time: 创建时间
  • update_ _time: 更新时间

因此，每个表里基本都有该两个字段哦！

4 总结

  • 表关系

  • 时间戳
